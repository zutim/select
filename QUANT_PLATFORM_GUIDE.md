# 统一量化系统 - 可视化平台使用指南

## 概述

本项目扩展了一个可视化量化分析平台，允许用户通过网页界面选择策略并进行量化分析。该平台与原有系统完全兼容，不改变现有逻辑。

## 新增功能

### 1. Web界面
- 直观的图形化界面
- 策略选择功能
- 实时数据分析
- 结果可视化展示
- 数据导出功能

### 2. 回测系统
- 完整的回测引擎
- 支持多策略回测
- 丰富的回测指标
- 净值曲线可视化
- 收益率分布图

### 3. API接口
- `/` - Web主页
- `/api/dates` - 获取可用交易日期
- `/api/run_strategy` - 运行策略分析
- `/api/status` - 系统状态查询
- `/api/backtest` - 运行回测
- `/api/backtest/metrics` - 获取回测指标

## 使用方法

### 启动系统

```bash
cd /path/to/unified_quant_system
go run main.go
```

系统将在 `http://localhost:8081` 上启动。

### Web界面操作

1. 打开浏览器访问 `http://localhost:8081`
2. 界面包含两个主要标签页：
   - **策略分析**: 选择日期和策略运行分析
   - **回测系统**: 设置回测参数运行历史回测

#### 策略分析标签页
- 选择日期和策略
- 点击"运行分析"按钮
- 在"分析结果"区域查看选股结果

#### 回测系统标签页
- 设置开始日期和结束日期
- 输入初始资金
- 选择策略类型
- 点击"运行回测"按钮
- 查看详细的回测报告，包括：
  - 总收益率、年化收益率
  - 波动率、最大回撤
  - 夏普比率、胜率
  - 净值曲线图
  - 日收益率分布图

### 策略说明

- **所有策略**: 运行全部选股策略
- **首板高开**: 选择首次涨停且高开的股票
- **首板低开**: 选择首次涨停且低开的股票
- **弱转强**: 选择从弱势转向强势的股票

## 故障排除

### 常见问题

1. **页面无法打开**
   - 检查服务是否正常启动
   - 确认端口8081未被占用

2. **数据加载失败**
   - 确保已有足够的历史数据
   - 检查数据目录权限

3. **策略运行失败**
   - 确认选择的日期有对应数据
   - 检查网络连接（如需实时数据）

4. **回测执行失败**
   - 确认日期范围有效
   - 确保初始资金为正值
