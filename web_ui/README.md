# 量化分析平台 - Web UI

这是一个可视化量化分析平台，允许用户通过网页界面选择策略并进行量化分析。

## 功能特性

- 直观的Web界面进行策略选择和分析
- 支持多种量化策略（首板高开、首板低开、弱转强等）
- 完整的回测系统，支持历史数据回测
- 实时系统状态监控
- 结果导出功能
- 数据可视化展示（净值曲线、收益率分布等）
- 丰富的回测指标（总收益率、年化收益率、最大回撤、夏普比率等）

## 接口说明

### 前端接口

- `GET /` - 主页，显示Web界面
- `GET /static/*` - 静态资源（CSS、JS、图片等）

### 后端API

- `GET /api/dates` - 获取可用的交易日期列表
- `POST /api/run_strategy` - 运行指定策略进行分析
  - 请求体: `{"date": "2024-01-01", "strategy": "all"}`
- `GET /api/status` - 获取系统状态信息
- `POST /api/backtest` - 运行回测
  - 请求体: `{"start_date": "2024-01-01", "end_date": "2024-12-31", "initial_capital": 1000000, "strategy": "all"}`
- `GET /api/backtest/metrics` - 获取回测指标说明

## 策略说明

1. **首板高开 (First Board High Open)** - 选择首次涨停且高开的股票
2. **首板低开 (First Board Low Open)** - 选择首次涨停且低开的股票  
3. **弱转强 (Weak to Strong)** - 选择从弱势转向强势的股票
4. **所有策略 (All)** - 运行所有策略

## 回测功能

- 支持指定时间段的历史回测
- 模拟真实交易环境（包含交易成本）
- 提供多种回测指标：
  - 总收益率
  - 年化收益率
  - 波动率
  - 最大回撤
  - 夏普比率
  - 胜率
- 可视化图表展示：
  - 净值曲线图
  - 日收益率分布图

## 使用方法

1. 启动主程序：`go run main.go`
2. 访问浏览器 `http://localhost:8081`
3. 在界面中选择日期和策略
4. 点击"运行分析"按钮执行量化分析
5. 查看结果并可选择导出
6. 或切换到"回测系统"标签页进行历史回测

## 技术架构

- 前端：HTML + CSS + JavaScript (Bootstrap框架, Chart.js图表库)
- 后端：Go语言 (Gin框架)
- 数据来源：复用原系统的数据管理和选股逻辑
- 回测引擎：自研的完整回测系统
